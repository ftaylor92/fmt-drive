About Google Drive APIs

There are many APIs for Google Drive:

  • Google Drive API for Android
    Pros:
    • Allows Android Apps to seemlessly Edit Google Drive Documents under the covers
    Cons:
    • Only Allows this Android App that created your Document to Edit it

  • Google Apps Script API
    Pros:
    • Your Document has all the resources of Google™ Drive (to print, copy, share, bookmark, view online, etc.)
    Cons:
    • Every interaction is done through a authorization certificate on the Web, so a browser window must pop up everytime a Apps Script is called

About Google Drive Document IDs


Google's security makes sure your saved document is private and secure only to you. When you create a ToDo list in your personal Google Drive, it will have a unique Google Drive Document ID, which you can set in the One-Tap Note app's "Google Drive Document ID" Preference if you'd like more control over which document to use for your ToDo list. You can create your ToDo list document now, if you prefer, and get its ID by clicking the Create Document Script.

You can find your Document Id by opening your Google Drive Document and viewing the long, strange id code in the URL.
