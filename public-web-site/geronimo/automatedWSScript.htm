Templated Web Services Deployment Script


This example contains:
  • a bash shell script to deploy a Web Service Client, a Web Service on server configurable through property files
  • two property files which should be configured depending on your enviorment
  • templates of J2EE Geronimo deployment descriptors, JWSDP cofiguration files, and source code which clearly show how each value is correlated though all the different files
  • ANT script which (un)deploys, packages and compiles the Web Services, and it changes templated files into useable Geronimo files
  • It does not use Eclipse WTP, but I suggest you do.  Regardless, this is a very comprehnsive analysis of Geronimo Deployment Descriptors and WSDL.
To install and use this:
  1. download and unzip the source code
  2. look at the %CONSTANTS% in the templates/variables.properites file in the templates directory for a clear correlation of deployment descriptor, corfiguration, and Java class values
  3. look at run-all.sh and build.xml to see all the steps neccessary to build a WSDL from Java Source, build Java Source from a WSDL, and to package and deploy a Web Service on Geronimo
  4. Once you run the ant template and ant template-wsdl ANT tasks from the run-all.sh script, you will have fully populated web/ and src/ folders with compilable Java code, JSP page, Web Service and Deployment Descriptors.  These along with Aaron Mulder's Geronimo Book, chapter 16, should teach you all you need to get started with Web Services on Apache Geronimo.  If not, email me for help.
To Deploy to Geronimo:
  1. install Java 1.4.2_10, Geronimo 1.0, JWSDP 2.0, and ANT 1.6.5, and cygwin 1.5 if on Windows
  2. change the templates/templates/variables.properties and build.properties files to conform to your environment.  The run-all.sh script needs some tweaking too
  3. start Geronimo and execute the run-all.sh script
