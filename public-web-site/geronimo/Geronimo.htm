Geronimo JavaWorld Source

These jars contain working source files for the 3rd(JSP jar), 4th(DB jar) and 5th(EJB jar) pages of JavaWorld's "A first look at Apache Geronimo" article.
The included files assume you are using Tomcat and a postgres Database Pool with Geronimo.  

postgres Database Pool:
To set up postgres to use Geronimo, I followed the example from section 6.3.1 of Aaron Mulder's Geronimo Book.  
Before I setup the datbase pool, I downloaded and installed postgres8.1.3 and put downloaded the postgresql-8.1-404.jdbc3.jar file into Geronimo's repository folder (under repository/postgres/jars).  I then created a candydb database and a candy table in the candydb database for test data in this example.

Ant deployment
Changing the "ghome" property in the build.xml file and typing "ant ear" from the command line, then deploying the ear file to Geronimo is all you'll have to do to use this example.
