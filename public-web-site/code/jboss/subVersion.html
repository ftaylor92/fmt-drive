SubVersion

Initialize
Install from http://subversion.tigris.org
check that ra-local exists with: svn --version
svnadmin create C:/development/CVSRepository

Add files
svn import C:/development/web/projectFolder file:///C:/development/CVSRepository -m "initial import"

Checkout (only checked out files are recognized by subVersion)
svn checkout file:///C:/development/CVSRepository projectFolder

create a .cvsignore file in projectFolder with the following contents:
build
dist
build.properties
----
svn propset svn:ignore -F ./projectFolder/.cvsignore ./projectFolder

to exclude folders and files from source control

svn status - to see differences
svn diff - differences in local version and source control version
svn commit - to commit changes to source control
svn update - to update source files to latest version
svn revert - reverts files to source control version
