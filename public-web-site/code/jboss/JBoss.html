JBoss Notes


  1. JBoss-mySQL connection

    1. Install JBoss, MySQL including ConnectJ mysql-connector-java-X-bin.jar file in <myconfig>/lib/

    2. Copy server/default to server/<myconfig>

    3. Add/change <myconfig>/conf/login-config.xml with xml.

    4. Add/change <myconfig>/deploy/mysql-ds.xml from <jboss>/docs/examples/jca

    5. Copy mysql-connector-java-3.2.0-bin.jar to <myconfig>/lib

    6. Create DB & tables in MySQL, then GRANT ALL PRIVILEGES on <db>.* to <jbossuser>@localhost IDENTIFIED by ‘<matt88-password>’ with GRANT OPTION;

    7. Run ./bin/run.bat –c <myconfig> to start, http://localhost:8080/jmx-console JBoss-system, type=server to stop.
