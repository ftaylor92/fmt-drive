How to Add a Counter to Your Web Page

  1. follow this template
  2. That's it. It's that easy!
  3. Works with any HTML web page (or JSP, PHP, etc), because it only uses JavaScript and REST
How it Works
  • It uses JQuery, so you have to add the JQuery library in you <head> section of your page, like this:
    <head>
    <script src="http://code.jquery.com/jquery-latest.min.js"></script>
    </head>
  • Then somewhere in you <body> section of your page, you have to put a <div id='conter'>. This is where the counter will be displayed:
    <div id='counter'></div>
  • To glue it all together and get the proper count of hits, add this javascript code:
    <script>
    $(document).ready(function() {
    $.ajaxSetup({ cache: false, contentType: "application/json; charset=utf-8", dataType:"json"});
    var counterUrl="https://fmt-bookmarks.herokuapp.com/rest/counter?site="+ encodeURIComponent(window.location.href);
    $.get(counterUrl, function(data, txtstatus, xbr) {
    $("#counter").html("<br/><br/><small><em>count: "+ data+"</em></small>");
    });
    });
    </script>
  • It identifies your web page by window.location.href, but feel free to replace that with any unique idenifier.
  • This code, <small><em>count: "+ data+"</em></small>, makes your counter look like this, count: 314, but feel free to change that to any style you prefer.
