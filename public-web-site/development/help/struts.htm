Struts

  • uses 5 custom JSP Tag libraries.
    • Struts-html, -bean, -logic, -nested, -template
  • MVC2 architecture.  Reusable framework for [non-]web GUIs: struts-config.xml, web.xml
    • View: JSP, ActionScript, ActionForm, DynaActionForm(code-less)
    • Actions: ActionMappings to business logic(Model), EJB, JavaBeans
      • Keep thread-safe
      • Don’t write new Action, just map to Bean
    • Controller: ActionServlet
