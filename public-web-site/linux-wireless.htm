How I got Belkin Wireless G (F5D7050) to work on Red Hat Enterprise Linux 5.2


In Three Steps:

1) Installed RHEL5.2 to get to kernel 2.6.8-92


2) Installed ndiswrapper 1.54 from http://sourceforge.net/projects/ndiswrapper/

tar xfvz <downloaded-file>

cd <exploded-directory>

make uninstall

make

su root

make install

copied 3 WIN2KXP drivers from Belkin Wireless G F5D7050 install disk

./ndiswrapper -i ../../../belkin-drivers/rt73.inf

./ndiswrapper -l to see driver and device are recognized


3) Installed wicd from http://wicd.sourceforge.net/

by following these directions: http://wicd.net/moinmoin/Wicd%20on%20FC9

yum -y install python python-devel wget (this failed for me, but didnâ€™t matter)

wget http://downloads.sourceforge.net/wicd/wicd-1.5.4.tar.gz

tar -xf wicd-1.5.4.tar.gz

cd wicd-1.5.4

python setup.py configure

python setup.py install

/sbin/service NetworkManager stop

/sbin/service NetworkManager start

/sbin/service NetworkManager stop

/sbin/chkconfig NetworkManager off

/sbin/chkconfig wicd on

/sbin/chkconfig wicd off

/sbin/chkconfig wicd on


afer restart, wicd appeared in top menu bar and I could connect to open wireless connections.
