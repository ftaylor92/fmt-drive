Linux Install

How to install Linux on Dell Inspiron 1100
  1. Install RedHat Enterprise Linux 4 (RHEL4)
  2. create ftaylor92 user and make taylors group. Use him, not root
  3. chown /usr/local/bin/ and /usr/local/lib to taylors group (chgrp taylors), and chmod bin 777
  4. put Terminal, LogOut, 4xDesktops, Firefox and Battery Monitor on top MenuBar
  5. put alias.sh, path.sh and env-vars.sh into ~/.dotfiles/ and call them in .bashrc w/. $HOME/.dotfiles/alias.sh...
